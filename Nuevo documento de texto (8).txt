Paso 2: Dar acceso a la vlan por su respectivo puerto (en cada switch)

Para la vlan administracion
int f0/1
switchport mode access
switchport access vlan 10

Para la vlan logistica
int f0/1
switchport mode access
switchport access vlan 20

--Asi pa todas las demas--
Para Ejecutivos y Clientes
int f0/1
switchport mode access
switchport access vlan 70

int f0/2
switchport mode access
switchport access vlan 60


Para Servidores
int range f0/1-3
switchport mode access
switchport access vlan 80

Paso 3: Crear las SVIs en cada Switch (A traves de la interfaz Vlan 99(Nativa))
en 
conf t
int vlan 99
ip address 10.5.81.81 255.255.255.240
exit
ip default-gateway 10.5.81.81

Paso 4: Realizar el Trunking (Switches)
en
conf t
int g0/1
switchport mode trunk
switchport trunk native vlan 99


Switch de 3ra capa o Switch Multicapa
ip routing
int range g1/0/2-8
switchport mode trunk
switchport trunk native vlan 99
